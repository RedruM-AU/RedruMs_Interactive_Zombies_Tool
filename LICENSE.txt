ANTIVIRUS FALSE POSITIVE NOTICE
=========================================================
This program may trigger antivirus warnings (Windows Defender, etc.).

This is a false positive. The tool was built in Python using
PyInstaller. It only reads public TikTok LIVE data and sends
keyboard input to Call of Duty games.

The installer automatically adds a Windows Defender exclusion
for the program folder.
